# Install package

```cmd
pip install git+https://github.com/openai/whisper.git
pip install setuptools-rust
```

```cmd
# on Windows using Chocolatey (https://chocolatey.org/)
choco install ffmpeg
```

# Start

```cmd
python v_t.py
```
